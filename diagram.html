Memory Layout Diagram

Cell Data:
WaterStored (4 bytes) Plant Level (4 bytes) Plant Type (4 bytes) Plant WaterStored (4 bytes) Plant SunStored (4 bytes)
4 bytes 4 bytes 4 bytes 4 bytes 4 bytes
Player Position:
Position i (4 bytes) Position j (4 bytes)
Time:
Time Step (4 bytes)

For a gridWidth x gridHeight grid:
Each cell occupies 20 bytes (5 fields x 4 bytes per field).
The entire grid occupies gridWidth * gridHeight * 20 bytes.
Player position occupies 8 bytes (2 fields x 4 bytes per field).
Time occupies 4 bytes.

Full Memory Layout:
Grid Data (gridWidth x gridHeight x 20 bytes) Player Position (8 bytes) Time (4 bytes)
